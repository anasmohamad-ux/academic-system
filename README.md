# Academic System

Full-stack Academic Management System

## Backend
- Laravel 12
- Sanctum Authentication
- Spatie Role & Permission
- Clean Architecture
- SOLID Principles

## Frontend
- React + Vite
- TypeScript
- TailwindCSS
- Role-based UI

## Features
- Authentication (Login/Register)
- Role-based Dashboards
- Admin User CRUD
- Course Management
- Enrollment System
